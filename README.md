# Gutenberg Library App

Gutenberg Library App, açık kaynaklı bir kitap kütüphanesi uygulamasıdır. Bu uygulama ile Gutenberg projesinden alınan kitap verilerini arayabilir, indirebilir ve okuyabilirsiniz.

## Özellikler

- **Arama:** Kitapları isme göre arayabilirsiniz.
- **Detaylı Bilgi:** Kitapların detaylı bilgilerini görüntüleyebilirsiniz.
- **Dark Mode:** Uygulamada karanlık modu etkinleştirebilir veya devre dışı bırakabilirsiniz.
- **Kitap İndirme:** Kitapları indirebilir ve yerel bir veritabanında saklayabilirsiniz.
- **Okuma Ekranı:** İndirilen kitapları okuma ekranında görüntüleyebilirsiniz.

## Kullanılan Teknolojiler

- [Android](https://developer.android.com/) - Uygulama geliştirmek için kullanılan mobil platform.
- [Kotlin](https://kotlinlang.org/) - Ana programlama dili.
- [Room](https://developer.android.com/topic/libraries/architecture/room) - Veritabanı yönetimi için kullanılan Android Jetpack bileşeni.
- [Retrofit](https://square.github.io/retrofit/) - HTTP istekleri için kullanılan kütüphane.
- [Glide](https://bumptech.github.io/glide/) - Resim yüklemek ve göstermek için kullanılan kütüphane.
